# Simple API for sensor project

## Description
To start the server use: `node server.js`

1. `/` - main page
2. `/data/:sensorID` - top 20 data entries for sensor ID `:sensorID`
3. '/api' - path for API

## Dependencies
1. Mongo DB
2. Node JS with npm (express, mongoose, body-parser)
3. bower (bootstrap, jquery)
